\begin{abstract}
In statistical inference, the frequentist approach treats parameters as unknown fixed constants, whereas the Bayesian framework views them as random variables following a probability distribution. While Monte Carlo methods allow for the computational approximation of these distributions, traditional centralized Markov Chain Monte Carlo (MCMC) becomes computationally prohibitive when dealing with large-scale or distributed datasets.

To address this, Consensus Monte Carlo offers a scalable alternative by partitioning data into shards, running separate MCMC chains locally, and combining the results to approximate the full posterior. This project implements a Consensus Monte Carlo framework using Independent Metropolis-Hastings (IMH) and Random Walk Metropolis-Hastings (RWMH). The study evaluates the accuracy of the consensus approach against a full-data Metropolis-Hastings baseline to determine the conditions under which the distributed method is an adequate substitute.

\vspace{6ex}

\noindent {\bf Keywords}: Bayesian Inference, Markov Chain Monte Carlo (MCMC), Metropolis-Hastings, Consensus Monte Carlo.

\end{abstract}
