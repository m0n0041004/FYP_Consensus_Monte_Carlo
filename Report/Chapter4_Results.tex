\chapter{Results and Analysis}

\section{Data Summary}

This chapter reports the empirical results for Bayesian logistic regression applied to the 2013 UK accident dataset \cite{ukdataservice7550,dft2025quality}. The response is binary accident severity, where serious/fatal accident severity is coded as 1 and is the event being modelled, while slight accident severity is coded as 0 and is the reference response class. The predictors are road type and speed limit.

The reported model settings give 82570 original observations and 82344 observations after preprocessing. The empirical analysis uses four independent chains for each final posterior summary and $K=10$ shards for Consensus Monte Carlo. Serious/fatal accidents account for 10970 observations, corresponding to 13.32\% of the analysis data, while slight accidents account for 71374 observations, corresponding to 86.68\%. The response distribution is shown in Table \ref{tab:response_distribution}.

The current implementation runs the full-data RWMH and full-data IMH samplers sequentially. In contrast, the CMC-RWMH and CMC-IMH shard computations use parallel workers, with the number of workers set equal to the number of shards. The recorded CMC runtimes therefore reflect the current parallel implementation and include cluster setup and worker overhead. The reported runtime values show shorter recorded wall-clock times for the CMC methods than for the corresponding full-data methods in this run. This observation should not be generalised as hardware-independent speedup, because runtime depends on the computing environment, parallel backend, cluster overhead, posterior combination cost, implementation details, and statistical efficiency.

\section{Summary of Findings}

The frequentist logistic regression provides an initial baseline, diagnostic checks, and proposal information for the Metropolis--Hastings samplers. Its $P$-values are reported descriptively, while the likelihood-ratio $G^2$ test assesses improvement over the intercept-only model and the GVIF values assess multicollinearity among the categorical predictors.

The full-data Random Walk Metropolis--Hastings and full-data Independent Metropolis--Hastings samplers produce similar posterior means, medians, standard deviations, credible intervals, and posterior odds ratios.
